## 我是谁？

小时候我是那种会被大人笑着说“这孩子想法跟别人不太一样”的小孩。别的小朋友沉迷动画片的时候，我趴在老爸那台嗡嗡响的旧电脑前面，追着他问为什么点这里屏幕就会变个样，里面到底藏了什么机关。那时候说不上来为什么着迷，就是觉得机器会“听指挥”这件事特别神奇。

后来兴趣越来越杂，但每一样都能让我安安静静琢磨好几个小时。中学进了计算机社团，算是第一次正经摸到编程的门，从写那种简陋的个人主页，到用 Python 让电脑自动帮我整理作业文件夹，每做出一个小玩意儿都能高兴一整天。

虽然大学没把计算机当成专业去读，但这份热爱一直没断过。它更像是我的秘密基地，学习累了、想法多了，就溜进来捣鼓点好玩的东西，不求多厉害，只求“这玩意儿居然真跑起来了”那一刻的快乐。

## 仓库使用指南

如果你打算在这里随便逛逛，友情提示几件事： 

1、你大概率会撞见某个写着“待完成”的计划文档  
2、有些项目我起了个酷酷的名字，结果里面就三行代码和一个空的 README，我自己也忘了当初想干嘛，但舍不得删  
3、还有几个仓库，纯粹是当时觉得“这个技术名字好听”就 clone 下来研究的，研究到一半就跑去吃西瓜了
